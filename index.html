华为云 IoT 通信与 OTA 升级系统
MQTTS 属性上报 · HTTPS 分片下载 · 断点续传 · CRC32 校验
MQTT mbedTLS FAL QSPI Flash
›
项目描述

基于华为云 IoT 平台的设备通信与远程升级系统。MQTT/MQTTS 协议实现设备认证与属性上报，OTA 升级采用 HTTPS 分片下载，支持断点续传与 CRC32 校验，确保固件安全可靠。

核心特性
  • MQTT/MQTTS 双模式连接，HMAC-SHA256 设备认证
  • 定时属性上报，OTA 运行时自动暂停上报
  • HTTPS 分片下载，支持断点续传
  • 智能 Flash 擦除策略，优化擦除时间
  • CRC32 完整性校验 + 下载重试机制（最多 3 次）
  • OTA 状态持久化，断电恢复

LVGL 图形界面系统
800×480 RGB565 · DMA2D 加速 · LTDC 防撕裂
LVGL 8.3 DMA2D LTDC GT911
›
项目描述

基于 LVGL v8.3.10 的嵌入式 GUI 系统，针对 STM32H743 优化。采用 SDRAM 双缓冲部分刷新，DMA2D 硬件加速颜色填充，LTDC 行中断同步彻底消除屏幕撕裂。lv_timer_handler() 自适应调度，静态时降低刷新频率减少 CPU 占用，动态时保持流畅响应。

核心特性
  • 800×480 RGB565 双缓冲部分刷新，SDRAM 8MB 显存
  • DMA2D 硬件加速，减轻 CPU 负担
  • LTDC 行中断同步，完全消除屏幕撕裂
  • GT911 电容触摸屏，中断模式读取
  • lv_timer_handler() 自适应调度，静态低频刷新降功耗，动态高频保流畅
  • 自定义 LVGL 任务管理，独立内存池

存储管理与硬件驱动
SDRAM 32MB · QSPI Flash · FAL 分区 · MPU
SDRAM QSPI FAL MPU
›
项目描述

完整的存储管理与内存调度子系统。链接脚本定义 5 区域内存布局，MPU 配置 4 区域独立 Cache 策略，多 RAM 混合调度让每种内存各司其职。修复了 SDRAM 时序关键 Bug（RCDDelay 160ns→30ns，提升 5.3 倍随机访问速度）。

核心特性
  • 链接脚本 5 区域内存布局：ROM(1.5MB) / DTCM(128KB) / AXI(512KB) / SDRAM(32MB) / SRAM4(64KB)
  • MPU 4 区域独立 Cache 策略：DTCM 零等待、AXI Write-Back、SRAM4 Non-Cacheable、SDRAM Write-Back
  • 多 RAM 混合调度：DTCM 系统堆+栈 → AXI memheap 绑定 LVGL 内存池 → SRAM4 固定块内存池供 ESP-Hosted DMA → SDRAM 帧缓冲+JPEG 环形缓冲
  • AXI memheap locked=RT_TRUE，DTCM 满时 rt_malloc 自动回退到 AXI
  • SRAM4 rt_mp_t 固定块内存池，零碎片，spinlock 保护，ISR 安全
  • D-Cache 一致性管理：DMA2D/MDMA/LTDC/JPEG 四路并发 Clean + Invalidate
  • SDRAM 时序优化，RCDDelay 修复（5.3x 提升）
  • QBoot 释放标志预留 4KB，智能 Flash 擦除策略
  • LTDC 像素时钟优化至 40MHz
